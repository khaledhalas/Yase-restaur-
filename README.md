
# Yase Restaurant - Premium Digital Menu

A premium, responsive digital menu presentation interface designed with modern web architecture patterns for **Yase Restaurant**.

## ✨ Features
* **Tailwind CSS Architecture:** Hand-tailored custom parameters extending classic design variants (`brandBrownDark`, `brandOliveDark`, `brandGold`).
* **Fluid Modular Layouts:** Fast responsive grid patterns adapt easily from widescreen desktop layouts down to fine mobile viewpoints.
* **Optimized View Controls:** Built-in lightweight state manager smoothly manages menu category switching without heavy framework overhead.
* **Fully Semantic & Accessible:** Built from the ground up using pure HTML5 tags and FontAwesome asset libraries.

## 🛠️ Tech Stack
* HTML5
* Tailwind CSS (via CDN configuration engines)
* JavaScript (ES6+ Native DOM Interactivity)
* FontAwesome Icons

## 📂 Assets Note
This interface utilizes background gradient textures pointing locally to `image_11.png` and `image_10.png`. Please make sure your design asset files exist in the root system alongside `index.html` to load background depth layers accurately.
